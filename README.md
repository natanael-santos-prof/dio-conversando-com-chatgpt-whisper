# 🎤 Assistente de Voz Inteligente com ChatGPT e Whisper

Este projeto foi desenvolvido como parte do desafio prático na plataforma **DIO**. Consiste em um assistente virtual capaz de ouvir a voz do usuário, transcrever o áudio, processar uma resposta inteligente e responder por voz.

## 🛠️ Tecnologias Utilizadas
* **Python**: Linguagem principal do projeto.
* **OpenAI Whisper**: Transcrição de áudio para texto de alta precisão.
* **OpenAI ChatGPT (gpt-4o-mini)**: Geração de respostas inteligentes e rápidas.
* **gTTS (Google Text-to-Speech)**: Conversão da resposta de texto em áudio.

## 🚀 Como Executar o Projeto Localmente

1. Certifique-se de ter o Python instalado.
2. Instale as dependências obrigatórias:
   ```bash
   pip install -r requirements.txt
   ```
3. Abra o arquivo `app.py` e insira sua chave pessoal da OpenAI na linha 9.
4. Execute o projeto no terminal:
   ```bash
   python app.py
   ```
